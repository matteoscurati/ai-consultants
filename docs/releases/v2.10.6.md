# Release v2.10.6

**Date:** 2026-05-02
**Type:** Patch — Premium model upgrades across 5 consultants + Kilo SIGPIPE fix
**Previous:** v2.10.5

## Highlights

- Codex, Cursor, Aider, DeepSeek, and Claude all bumped to newer premium defaults
- Aider switches provider to NVIDIA Nemotron 3 Super (free tier) via OpenRouter

Aider moves off OpenAI Codex to NVIDIA's Nemotron 3 Super 120B (A12B-active MoE) on the OpenRouter free tier, lowering cost-per-consultation for collaborative coding workflows.

### Kilo SIGPIPE Fix

`query_kilo.sh` previously truncated output via `echo "$CONTENT" | head -5 | ... | head -c 200`. Under `set -o pipefail`, `head -c` closing the pipe before the upstream `sed` finished writing produced a SIGPIPE that aborted the entire script. Replaced the trailing `head -c` with bash parameter expansion (`${var:0:N}`), which truncates safely without involving the pipeline.

## Breaking Changes

None. Existing `CLAUDE_MODEL`, `CODEX_MODEL`, `CURSOR_MODEL`, `AIDER_MODEL`, and `DEEPSEEK_MODEL` overrides continue to work — only the *defaults* changed.

If you were relying on the short Claude aliases (`opus-4.6`, etc.) in scripts, those still work as overrides through the CLI; the change only affects what's selected when no override is present.
