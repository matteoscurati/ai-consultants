### Kilo CLI Fix

Resolved a long-standing issue where the Kilo consultant would hang for the full timeout (180s) and return empty or ANSI-only output. Root causes:

1. **Non-TTY hang**: Kilocode CLI hangs when the query is passed as a CLI argument and stdout is redirected. Fixed by passing the query via stdin instead.

2. **Wrong provider auto-detection**: Kilocode auto-detects API providers from environment variables. When other consultants' API keys (e.g., `MINIMAX_API_KEY`) were exported, kilocode would pick non-functional models like `MiniMax-M2.7-highspeed`. Fixed by unsetting foreign API keys via `env -u` before invocation.

3. **Output cleaning**: Collapsed 5-stage pipe (`sed | sed | tr | grep | grep`) into a single `sed` invocation. Fixed overly aggressive markdown fence filter that would strip opening code fences like `` ```typescript `` — now only strips standalone `` ``` `` lines.
