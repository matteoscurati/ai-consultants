## Highlights

- Grok premium upgraded from `grok-4.20-0309-reasoning` to `grok-4.3`
- ~75% cheaper input, ~83% cheaper output, with a 1M-token context window
- xAI explicitly recommends migrating all callers to grok-4.3

## What's New

### Grok 4.3 Upgrade

xAI released Grok 4.3 on April 30, 2026 as their flagship model, replacing the previous reasoning-focused `grok-4.20-0309-reasoning`. The new model is positioned as both the most intelligent and the fastest in the lineup, with aggressive pricing.

| Tier | Before (v2.10.6) | After (v2.10.7) |
|------|-------------------|------------------|
| Premium | grok-4.20-0309-reasoning ($5.00/$15.00 per M) | grok-4.3 ($1.25/$2.50 per M) |
| Standard | grok-3 ($3.00/$9.00 per M) | grok-3 (unchanged) |
| Economy | grok-3-mini ($1.00/$3.00 per M) | grok-3-mini (unchanged) |

The premium tier moves from $20.00 combined per million tokens (input + output average) to $3.75 — a roughly 5x reduction at typical query shapes. The 1M-token context window is unchanged from the prior reasoning variant.

**Note on tiered pricing:** xAI bills requests exceeding 200k input tokens at a higher rate. For very long-context queries, consult the xAI Console for the exact tiered pricing.

The standard and economy tiers continue to use `grok-3` and `grok-3-mini` respectively. The previous premium model `grok-4.20-0309-reasoning` was moved to the legacy section of `docs/cost_rates.json` so cost calculation continues to work for any historical responses or pinned overrides.

## Breaking Changes

None. Existing `GROK_MODEL` overrides continue to work — only the default changed.
