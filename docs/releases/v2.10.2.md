# Release v2.10.2

**Date:** 2026-03-19
**Type:** Patch — MiniMax M2.7 model upgrade
**Previous:** v2.10.1

## Highlights

- MiniMax upgraded from M2.5 to M2.7 (released 2026-03-18)
- Premium and standard tiers now use MiniMax-M2.7
- Economy tier uses MiniMax-M2.5 (previous default)
## What's New

### MiniMax M2.7 Model Upgrade

MiniMax released M2.7 on March 18, 2026. This patch updates all MiniMax tier assignments:

| Tier | Before (v2.10.1) | After (v2.10.2) |
|------|-------------------|------------------|
| Premium | MiniMax-M2.5 | MiniMax-M2.7 |
| Standard | MiniMax-M2.1 | MiniMax-M2.7 |
| Economy | MiniMax-M2.5-highspeed | MiniMax-M2.5 |

Default model (`MINIMAX_MODEL`) is now `MiniMax-M2.7`.

## Breaking Changes

None.

## Upgrade Guide

```bash
# If installed via git clone
cd ~/.claude/skills/ai-consultants && git pull
```

No configuration changes required. Existing `MINIMAX_MODEL` overrides continue to work.
