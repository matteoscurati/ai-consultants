# Release v2.17.0

**Date:** 2026-06-24
**Type:** Minor — model catalog refresh (all agents, all tiers) + cost rates
**Previous:** v2.16.0

## Highlights

- **Every consultant's models refreshed for June 2026**, across all three tiers (premium / standard / economy), with updated per-token cost rates.
- CLI-based agents were verified by **querying the installed CLIs** (e.g. `agy models`, `agent --list-models`); API/provider models and pricing were **researched against official sources**.
- Superseded model IDs are retained in the cost catalog for historical and pinned-override lookups — nothing breaks for cached results.
- Fully overridable: set a per-agent `*_MODEL` env var (or a tier preset) to pin any model.

### Notable removals / deprecations

- `gpt-5.3` (standalone) and `gpt-4o-mini` are superseded in the OpenAI lineup.
- DeepSeek's `deepseek-chat` / `deepseek-reasoner` are being deprecated (2026-07-24) and now route to the V4 line; defaults moved to `deepseek-v4-pro` / `deepseek-v4-flash`.

### Cost rates

Rates were refreshed to verified provider pricing (per-1K, the runtime contract). Several stale entries were corrected — most importantly `gpt-5.5`, which had been understated. `docs/COST_RATES.md` was regenerated to match `cost_rates.json` exactly. Two entries (`glm-5.2`, `qwen3.7-max`) are estimates mirroring the prior version's pricing and are flagged to be re-verified when official numbers publish.

## Breaking Changes

None. Defaults change, but every model is overridable via env vars, and superseded IDs remain in the cost catalog for historical lookups.

## Upgrade Guide

```bash
# If installed via git clone
cd ~/.claude/skills/ai-consultants && git pull

# If installed via curl | bash
curl -fsSL https://raw.githubusercontent.com/matteoscurati/ai-consultants/main/scripts/install.sh | bash
```

No configuration changes required. To pin a specific model, set the per-agent env var, e.g. `export CODEX_MODEL=gpt-5.3-codex` or `export OLLAMA_MODEL=qwen2.5-coder:32b`.
